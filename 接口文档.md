### 接口文档

#### 1、上传文档

请求：

```python
	url = "http://121.62.23.178:51988/upload-file"
	response = requests.post(
		url=url,
		files={'file': open('xxx.pdf','rb')}
	)
    
    
```

返回：

```json
{
    'pdfId': 'pdf_47d7f91f-a518-4fe9-a12b-c83f99b7a4b5', 
    'message': 'Success', 
    'code': 200, 
    'time': '2023-12-04 17:25:08'
}

pdfId: 上传pdf的ID
```
